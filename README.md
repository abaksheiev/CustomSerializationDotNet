This repository contains the code for the first video in the "Custom Serialization in .NET" series.

## Description
In this video, we cover:
- Default JSON serialization in .NET
- When default serialization is not enough
- Writing a simple custom JsonConverter
- Serializing and deserializing objects
- Viewing results in the console

## How to run
1. Open the folder `Video1_BasicConverter` in Visual Studio or any C# IDE.
2. Run `Program.cs`.
3. Observe console output:
    - Serialized JSON
    - Deserialized object

## Notes
- This code uses System.Text.Json
- For demonstration purposes, the `Money` type is used
- Next videos will expand with polymorphism and versioning
